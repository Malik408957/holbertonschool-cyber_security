# 0x07. Nmap Post Port Scan Scripting

## Description

This project focuses on using the Nmap Scripting Engine (NSE) to perform post port-scan analysis.
The goal is to automate information gathering by running Nmap default NSE scripts against a given target.

## Task 0: Default NSE Scan

The script `0-nmap_default.sh` runs the default Nmap NSE scripts on a target host provided as a command-line argument.
